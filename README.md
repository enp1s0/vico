# Vico - a simple job queue relying on 'tmux'.

'vico' means 'queue' in Esperanto.

## Installation
1. Clone this repository
```bash
git clone https://github.com/enp1s0/vico.git
```

2. Set `$PATH`
```bash
export PATH=/path/to/vico/bin:$PATH
```

## Usage
### Submit a job
```bash
qsub job.sh
```

### Check job status
```bash
qstat
```
